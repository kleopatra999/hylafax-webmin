Connessioni al server HylaFax
Inserire una o più regole, con le seguenti specifiche:

  • immettere un username, o lasciare vuoto e selezionare 'tutti';
  • immettere un nome host, o un indirizzo IP completo, o un indirizzo IP parziale (nel qual caso si specifica una sottorete), o un nome di dominio parziale.
    Esempi:
    192.168.0.1: solo questo IP
    10.0.0.: l'intera sottorete 10.0.0.x
    tizio.caio.it: questo host
    .caio.it: l'intero dominio caio.it
