Conexiones con el servidor HylaFax
Rellene una o más reglas de la siguiente forma:

  • indique un usuario, o déjelo en blanco y elija 'todos';
  • escriba el nombre de un ordenador, una dirección IP completa, una dirección IP incompleta (una subred), o un nombre de dominio (completo o parcial).
    Ejemplos:
    192.168.0.1: sólo esta IP
    10.0.0.: toda la subred 10.0.0.x
    john.doe.com: este ordenador
    .doe.con: todo el dominio doe.com
